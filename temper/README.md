# How to use `temper` ?

```bash
git clone https://github.com/bitplane/temper.git
cd temper
make
sudo chown root:root ./temper
sudo chmod u+s ./temper
mv ./temper /usr/local/bin/temper
```

You can apply `temper.patch` if it needs to get the temperature only.
